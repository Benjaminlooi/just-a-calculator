# Popup Design Revamp Plan — "Just a Calculator"

## Goal

Full visual redesign of the extension popup: proper light/dark theming, improved typography, visual hierarchy with the tally counter as the hero, subtle animations, and a polished satirical personality.

---

## 1. Design System

| `--border` | `#262626` | Subtle borders |
| `--text` | `#e5e5e5` | Primary text |
| `--text-muted` | `#737373` | Secondary/muted text |
| `--text-faint` | `#525252` | Footer, citations |
| `--accent` | `#646cff` | Arrows, replacements, links |
| `--accent-dim` | `#646cff33` | Accent background tint |

### Typography

- **Font import**: Google Fonts — Space Grotesk (500, 700) + Inter (400, 500, 600) via `<link>` in index.html
- **Title**: Space Grotesk 700, 1.25rem (20px)
- **Counter number**: Space Grotesk 700, 2.5rem (40px), tabular-nums
- **Counter label**: Inter 500, 0.75rem (12px), uppercase, letter-spacing 0.05em
- **Body text / subtitle**: Inter 400, 0.875rem (14px)
- **Examples**: Inter 400, 0.8125rem (13px)
- **Quote**: Inter 400 italic, 0.8125rem (13px)
- **Footer**: Inter 400, 0.6875rem (11px)

### Layout (320px wide)

```
┌──────────────────────────────┐
│  [icon]  AI → Calculator     │  ← header row, icon + title
│  subtitle text               │
├──────────────────────────────┤
│  ┌────────────────────────┐  │
│  │      1,247             │  │  ← hero counter card
│  │  HYPE TERMS REMOVED    │  │
│  └────────────────────────┘  │
├──────────────────────────────┤
│  AI → Calculator             │  ← examples, compact list
│  artificial intelligence →   │
│    Artificial Calculator     │
│  ... (scrollable, max-h)     │
├──────────────────────────────┤
│  "quote text..."             │  ← quote
│  — attribution               │
├──────────────────────────────┤
│  footer text                 │

```

- Padding: 20px horizontal, 16px vertical sections
- Section gaps: 16px between major sections
- Counter card: full-width, border-radius 12px, border 1px solid var(--border), padding 16px
- Examples: max-height ~200px with overflow-y auto, custom scrollbar

## 2. File Changes

### `entrypoints/popup/index.html`
- Add Google Fonts `<link>` for Space Grotesk (500;700) and Inter (400;500;600) in `<head>`
- Keep existing meta tags and structure

### `entrypoints/popup/style.css`
- Replace hardcoded colors with CSS custom properties on `:root`
- Add `@media (prefers-color-scheme: light)` block to swap all `--*` tokens
- Update body: remove `min-height: 100vh` (popup doesn't need it), keep `margin: 0`
- Update font-family to include Space Grotesk as first choice for headings
- Add `prefers-reduced-motion` media query to disable transitions
- Remove button/anchor styles that don't apply to the popup

### `entrypoints/popup/App.css`
- Full rewrite using CSS custom properties
- **Popup container**: 320px width, padding 20px, border-radius 0 (browser provides popup chrome)
- **Header**: Flex row with small calculator icon (inline SVG) + title. Title uses Space Grotesk.
- **Subtitle**: Muted color, slightly smaller
- **Counter card**: `background: var(--bg-card)`, `border: 1px solid var(--border)`, `border-radius: 12px`, centered text, large Space Grotesk number, uppercase tracking label
- **Examples list**: Left-aligned, compact rows. Each row: strikethrough original → accent replacement. Max-height with scroll. Alternating subtle row backgrounds.
- **Quote**: Italic, muted, with a left border accent line (2px solid var(--accent))
- **Footer**: Smallest text, faint color, centered
- **Transitions**: All color transitions 150ms ease on interactive elements

### `entrypoints/popup/App.tsx`
- Add a small inline SVG calculator icon component for the header
- Keep the same data flow (useState + useEffect + storage watch)
- Keep the same EXAMPLES array
- Restructure JSX to match new layout (header row, counter card, scrollable examples, quote with border, footer)
- Add `role="status"` and `aria-live="polite"` on the counter for accessibility
- Keep singular/plural logic for the label

---

## 3. Animations

- **Counter number**: `transition: opacity 150ms ease` on value change (subtle fade on update)
- **Counter card**: `transition: background-color 150ms ease, border-color 150ms ease` for hover
- **Examples rows**: `transition: background-color 150ms ease` on hover
- **All transitions**: Wrapped in `@media (prefers-reduced-motion: no-preference)` — disabled when user prefers reduced motion
- No scale transforms, no bouncy animations — keeping it clean and professional

---

## 4. Accessibility

- Counter gets `role="status"` + `aria-live="polite"` so screen readers announce updates
- All text meets WCAG AA contrast (4.5:1 minimum) in both modes
- `prefers-reduced-motion` disables all transitions
- Focus-visible outlines on any interactive elements
- Semantic HTML: `<h1>` for title, `<ul>/<li>` for examples, `<blockquote>` for quote
