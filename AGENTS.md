# AGENTS.md

## What this is

WXT + React browser extension ("Just a Calculator") that replaces AI buzzwords with "Calculator" on web pages. Chrome (MV3) and Firefox (MV2). Small single-package project — no monorepo.

## Commands

```bash
pnpm install          # postinstall runs `wxt prepare` to regenerate .wxt/ types
pnpm dev              # Chrome dev server (hot reload)
pnpm dev:firefox      # Firefox dev server
pnpm build            # Chrome production build
pnpm build:firefox    # Firefox production build
pnpm compile          # Type-check only (tsc --noEmit)
pnpm zip              # Pack Chrome extension
pnpm zip:firefox      # Pack Firefox extension
```

**Run `pnpm compile` before committing** — there is no separate lint step. TypeScript is the only static check.

## Key paths

| Path | Purpose |
|------|---------|
| `entrypoints/content.ts` | Content script — walks text nodes, regex-replaces AI terms, MutationObserver for dynamic content |
| `entrypoints/background.ts` | Background script |
| `entrypoints/popup/` | React popup UI (shows hype tally) |
| `wxt.config.ts` | Extension manifest, permissions, WXT config |
| `.wxt/` | Generated by `wxt prepare` — do not edit directly, gitignored |
| `.output/` | Build output, gitignored |
| `tsconfig.json` | Extends `.wxt/tsconfig.json` (generated) — edit compilerOptions here, not in `.wxt/` |

## Gotchas

- `tsconfig.json` extends `.wxt/tsconfig.json`. After a fresh `pnpm install`, `.wxt/` is regenerated — don't worry if it doesn't exist yet, just run `pnpm install`.
- The only permission requested is `storage`. No network, no tabs. The tally is stored under `storage.local` key `hypeRemoved`.
- Content script skips `<script>`, `<style>`, form fields, and other non-visible text nodes. Tally flushes every 1.5 seconds.
- Firefox build suppresses the `firefoxDataCollection` warning via `wxt.config.ts` `suppressWarnings`.
